Agrega fácilmente el seguimiento de Yandex Metrica en tu web y activa con un clic las funciones Webvisor, Scroll Map y Análisis de Formularios.

== Description ==

Inserta de forma rápida y sencilla el seguimiento de Yandex Metrica en todas las páginas de tu web y activa las funciones Webvisor, Scroll Map y Análisis de Formularios en un clic.

= Características =
* Compatible con las versiones más recientes de Wordpress.
* Sencillo y fácil de configurar, en apenas unos segundos dispondrás del seguimiento de Yandex Metrica en tu web. 
* Posibilidad de activar/desactivar opciones adicionales como Webvisor, Scroll Map y Análisis de Formularios.
* Ocupa menos de 20 Kb para interferir al mínimo en el rendimiento de tu web.
* Todos los detalles del plugin y un tutorial en <a href="https://www.labschool.es/yandex-metrica-y-google-analytics/" target="_blank">Lab School</a>


== Installation ==

1. Puedes:
 * Subir la carpeta al directorio */wp-content/plugins/* vía FTP. 
 * Subir el archivo ZIP completo vía *Plugins -> Añadir nuevo -> Subir* en el Panel de Administración de tu instalación de WordPress.
 * Buscar **Pixel de Yandex Metrica** en el buscador disponible en *Plugins -> Añadir nuevo* y pulsar el botón *Instalar ahora*.
2. Activar el plugin a través del menú *Plugins* en el Panel de Administración de WordPress.
3. Configurar el plugin dentro de *Ajustes -> Yandex Metrica*.
4. ¡Eso es todo!
